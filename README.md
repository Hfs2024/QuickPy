# 🐍 QuickPy

**QuickPy** is the ultimate browser-based Python playground! Copy, paste, and execute scripts instantly while mastering the language through interactive lessons and quizzes.

---

## 🚀 Features
* **Snippet Testing:** Copy-paste Python snippets and run them directly in your browser.
* **Save Your Work:** Keep track of your favorite snippets for later use.
* **Interactive Learning:** Learn Python step-by-step with guided lessons.
* **Challenge Yourself:** Test your knowledge with built-in Python quizzes.

## ⚠️ Warning
> **Note:** QuickPy is currently in **Beta**. We are working hard to improve the experience, so some features might still be under development. Stay tuned for updates! 🛠️
